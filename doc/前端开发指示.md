## 任务描述

你需要根据“前端API接口文档.md”和“前端开发与测试交付文档.md”的指示，进行一个个人博客页面的开发设计，先形成一份前端设计文档，保存在本地，然后进行开发。开发完成后进行本地测试。该项目的设计文档为“数据模型设计.md”和“API设计.md”。最初的要求在其他文档里的"博客网站设计（...）.md"。

## 技术栈

根据以上内容完成前端设计，技术栈参考：

```text
前台好看：
Tailwind 自定义首页、文章页、项目页

后台好做：
Ant Design 快速完成表格、表单、上传、设置页

文章链路完整：
@uiw/react-md-editor 写 Markdown
react-markdown 渲染 Markdown

前后端分离清晰：
Axios 调 Spring Boot API
React Router 管理页面跳转

工程感强：
React + TypeScript + Vite 是现代前端项目的标准组合
```

部署：
和后端一样，采用docker部署
